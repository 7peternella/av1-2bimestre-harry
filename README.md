# AV1 - 2º Bimestre - Harry Potter

Este projeto é uma página de listagem que consome dados da API pública de Harry Potter (https://hp-api.herokuapp.com/api/characters).

## Funcionalidades

- Consumo de API com fetch e async/await
- Manipulação de dados JSON
- Exibição dinâmica no DOM
- Layout responsivo com Bootstrap
- Feedback de carregamento
- Tratamento de erros

## Estrutura do Projeto

- `index.html`: Página principal
- `css/style.css`: Estilos personalizados
- `js/script.js`: Lógica JavaScript

## Como executar

Abra o arquivo `index.html` em um navegador web.
